##  介绍
`neoui-kero-mixin`对适配层的公共逻辑进行提取，适配层可根据需要添加对应的mixin



## 安装

首先选择一个目录，克隆项目文件:

```
$ git clone git@github.com:iuap-design/neoui-kero-mixin.git
```

然后进入目录安装依赖：

```
$ cd neoui-kero-mixin && npm install
```

接下来，执行下面的命令产出：

```
$ npm run product
```
